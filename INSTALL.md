### Linux & macOS

Run the following command in your terminal:

```bash
curl -fsSL https://raw.githubusercontent.com/akshayejh/green-bot/main/scripts/install.sh | sh
```

### Windows

Run the following in PowerShell:

```powershell
iwr -useb https://raw.githubusercontent.com/akshayejh/green-bot/main/scripts/install.ps1 | iex
```

## ðŸ“¦ Manual Installation

If you prefer not to use the automated scripts, you can download the latest version directly from our [Releases Page](https://github.com/akshayejh/green-bot/releases/latest).

#### macOS

1. Download the `.dmg` file.
2. Open it and drag "Green Bot" to your Applications folder.

#### Windows

1. Download the `.msi` or `.exe` installer.
2. Run the installer and follow the prompts.

#### Linux

1. Download the `.AppImage` file.
2. Make it executable: `chmod +x Green_Bot_*.AppImage`
3. Double-click to run.
